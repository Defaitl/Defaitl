<h2 align="center">About Me</h2>

<p align="center" width="80%">
Занимаюсь backend разработкой на Python: проектирую API, работаю с базами данных, кэшированием и очередями задач, слежу за тем, чтобы система выдерживала реальную нагрузку, а не только тестовый прогон. Пишу асинхронный код, покрываю его тестами, настраиваю автоматическую сборку и деплой.
</p>

<p align="center" width="80%">
Отдельно специализируюсь на LLM инжиниринге: строю RAG пайплайны с нуля, от обработки документов до векторного поиска и мультиагентных сценариев. Много работаю с prompt engineering: проектирую системные промпты для многошаговых сценариев, тестирую и итеративно улучшаю их под конкретные задачи, снижаю количество галлюцинаций через структурированные инструкции и проверку релевантности контекста. Сталкивался с задачами ускорения поиска в разы и автоматизации генерации контента через LLM.
</p>

<p align="center" width="80%">
Работал с высоконагруженными интеграциями: внешние API, обход лимитов, устранение узких мест в базе данных. Из конкретики: сокращал время обработки задач в десятки раз, снижал нагрузку на базу кэшированием, повышал точность поиска в RAG системах до 90%+.
</p>
